#Agenda And Agendash DEMO

本地运行，需要安装mongodb，推荐直接安装docker和docker-compose.运行之后直接查看`http://localhost:8080/agendash/#`

```shell
#直接看效果
docker-compose up

OR
#本地运行
npm install
npm test
npm start
```
